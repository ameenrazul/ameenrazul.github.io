🛑 Conclusion

This is a confirmed case of malware delivery via a poorly protected HTTP directory listing.

  • A user accessed the malicious site jx2-bavuong[.]com
  • Downloaded a suspicious .exe file
  • File has been flagged as a Trojan by multiple antivirus engines
  • Domain and server exhibit characteristics of malicious infrastructure

📌 Recommendations

  1. Block the domain/IP: jx2-bavuong[.]com, 141[.]164[.]41[.]174
  1. Scan affected systems for indicators of compromise (IOCs).
  1. Harden web filtering policies to prevent access to suspicious directories.
  1. Enable executable file type blocking over HTTP where not needed.
  1. Educate users about dangers of curiosity-driven exploration of unsecured web directories.
