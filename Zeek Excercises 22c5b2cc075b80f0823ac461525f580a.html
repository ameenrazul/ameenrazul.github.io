this one was a doozy, so there are a lot of dns queries which are liek xxxx.cisco update.com, so i had to find a way to check unique domain ones, not just unique ones , so i used cut query to cut the main part, then i used rev to reverse the cut out value, then i used cut to cut only values by using . as delimited twice, which will give me domain , then i used rev to reverse it back to normal, then sorted it and uniq it , then sort -nr to get the numebr

cat dns.log | zeek-cut query | rev | cut -d '.' -f 1-2 | rev | sort | uniq -c | sort -nr
