There is no way to do it via USB. So what's their other option?

Windows creates Jumplists to track recently accessed files, folders, or applications. These are especially useful in forensic investigations to piece together user activity on a system.

To get started:

  1. Open Command Prompt and type JLECmd.exe -h to see the list of available options.
  1. Use the f flag with the following command:
JLECmd.exe -d C:\Users\THM-RFedora\Desktop\kape-results\C

This command will process all the jumplists for the system. Scroll through the output until you find the “desktop” section. Notice the interaction count is 2, indicating the number of times the desktop was accessed. From here, you’ll find the answers to the relevant questions.
